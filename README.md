# ATM-SIMULATOR-PYTHON
-------- Python ATM System --------

 About:

- This is a simple ATM simulation project built using Python
- It allows users to check balance, deposit money, and withdraw money
- Data is stored in a file, so balance is saved even after closing the program

---

Features:

- Check current balance
- Deposit money
- Withdraw money
- Data saved permanently using file system
- Simple menu-based interface

---

Concepts Used:
 
-- Functions
- Loops ("while")
- Conditions ("if-elif-else")
- File Handling
- JSON module

---

File Used:

- "balance.json" → stores user balance

---

How to Run:

- Install Python
- Copy the code into a ".py" file
- Run the program
- Follow the menu options

---

Example Flow:

- Program starts → loads previous balance
- User selects option (deposit/withdraw/check)
- Updated balance is saved on exit

---
